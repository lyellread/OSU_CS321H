# Run Programs

## Python: `password_acceptor.py`

Requires: 
```
python3
```

Run:
```sh
./password_acceptor
```

## Bash Script: `email_acceptor.sh`

Run:
```sh
./email_acceptor.sh <email>
```

## C: `full_name_acceptor.c`

Requires:
```sh
gcc
[make]
```

Compile:
```sh
make
```

Run:
```sh
./full_name_acceptor
```
